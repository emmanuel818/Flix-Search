FLIX SEARCH

Flix search is a web application that will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of thier favorite movies.

Business Logic URL HTTP Method Request body data format Response body data format
Return a list of ALL movies to the user /movies GET None A JSON Object holding data about all movies
Return data about a single movie by title /movies/[title] GET None A JSON Object holding data about a sinlge movie, containing description, director, image URL, whetherit's featured or not
Return data about a genre(description) by name/title (e.g., "Thriller") /movies/[genre] GET None A JSON object holding a data about the genre
Return data about a director by name /movies/[director] GET None A JSON object holding data about a director, containing, bio, birth year, death year
Allow new users to register /register POST A JSON object holding data about the user to add should be like: A JSON object holding data about the user that was added

Allow users to add a movie to their list of favorites /user/[movie] PUT None A text message indicating that a movie was added to the list of favorites
Allow users to remove a movie to their list of favorites /user/[movie] DELETE None A text message indicating movie was succesfully removed from list
Allow existing users to deregister /user/[username] DELETE None A text message indicating that user has succesfully deregistered
